这一个简易的匠魂合金分离模组
添加了一种材料（月季铁）、两个可以存储材料的容器，以及最重要的——合金分离器。
合金分离器会将其上方容器中的合金材料缓慢分解为合金的制造材料，并将其输出到下方的容器中。
配方读取自任何匠魂的AlloyRecipe，加载时机为“首个方块实体实例化时”，因此理论上与其他附属之间可以兼容。

'''注意事项：'''
1.模组会自动获取在数据包中写明的反合金配方可以明确让分离器按照配方来分离。
1.1.如果一种合金存在多个配方（如Tinker's Thinking添加的钴+阿迪特=玛玉灵），本模组只会读取其中一者的合金配方（大部分情况下是本家的配方）。
1.2.鉴于以上原因，整合包作者建议使用时添加相应的反合金配方来配置对应合金该如何分解，本模组不提倡靠此模组刷取物品的行为。
2.自动生成的反合金配方的液体消耗量始终会除以10，如果不为10的倍数可能会出现液体损耗问题。
3.匠魂本家的配置目前还不清楚如何读取，因此不进行考虑。
3.1.鉴于以上原因，目前“熔融下界合金”的反合金配方始终仅返还4:2的残骸:黄金。
4.目前无法读取到合金配方的催化剂配置，原因不明。
