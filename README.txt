这一个简易的匠魂合金分离模组
添加了一种材料（月季铁）、两个可以存储材料的容器，以及最重要的——合金分离器。
合金分离器会将其上方容器中的合金材料缓慢分解为合金的制造材料，并将其输出到下方的容器中。
配方读取自任何匠魂的AlloyRecipe，加载时机为“首个方块实体实例化时”，因此理论上与其他附属之间可以兼容。

注意事项：
1.如果一种合金存在多个配方（如Tinker's Thinking添加的钴+阿迪特=马玉玲），本模组会尝试更早创建的合金配方。整合包作者建议使用前自行确认有无此类配方。
2.反合金配方的液体消耗量始终会除以10，如果不为10的倍数可能会出现液体损耗问题。
3.暂时还不能处理匠魂配置中的“cheaper_netherite_alloy”配置，始终按4:2=1的那个配方返还材料。
